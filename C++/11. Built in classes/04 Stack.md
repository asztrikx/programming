> FIFO / First In First Out structure  
> You can only take out the lastly pushed item creating a reverse order  

# Declaration
```c++
#include <stack>
```
```c++
stack<type> st;
```

# Time Complexity
|  Property  | Time |
| ---------- | ---- |
| empty()    | 1    |
| pop()      | 1    |
| push(type) | 1    |
| top()      | 1    |
