> Double linked list  
> Much slower than vector  

# Time Complexity
|  Property  | Time |
|------------|------|
| front      | 1    |
| back       | 1    |
| resize     | N    |
| assign     | N    |
| pop_front  | 1    |
| pop_back   | 1    |
| push_front | 1    |
| push_back  | 1    |
| insert     | 1    |
> `erase` returns iterator to next element
